# Licensing

Elder Council is **dual-licensed**:

- **Code → Apache-2.0** ([LICENSE](../LICENSE)). Permissive, with an explicit patent grant — the
  norm enterprises pre-approve, so adopting the harness is frictionless.
- **Documentation & methodology → CC BY 4.0** ([LICENSE-DOCS](../LICENSE-DOCS)). The method, the
  council definitions as prose, and the docs may be reused and adapted with attribution.

Every source file carries an `SPDX-License-Identifier` header so the licence is unambiguous per file.

## Why the split

The harness is local-first and offline-capable; a copyleft licence on the code would buy adopters
nothing and add friction. Apache-2.0's patent grant is what corporate legal teams look for. The docs
use CC BY 4.0 so the *methodology* spreads freely (that is the point) while keeping attribution.
